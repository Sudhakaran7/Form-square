Given a rectangle of size n x m, find the minimum number of integer-sided squares that tile the rectangle.
For example, Consider this example, n=2, m=3,

Input Description:
n and m of the rectangle.(1<n,m<100)

Output Description:
Print the minimum number of integer-sided squares that tile the rectangle.

Sample Input:
2 3

Sample Output:
3

Explanation:
Consider the example in question for explanation.
